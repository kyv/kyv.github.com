Plugin Audio Ikiwiki

Hace poco tiempo hicimos un plugin de audio para ikiwiki. Esto se encuentra en nuestra bifurcación de ikiwiki. El plugin vive en la carpeta Ikiwiki/Plugin. Pero depende también de un underlay en que hemos copiado el 360° player desde el proyecto soundmanager2.

Para ver el plugin en acción visita flujos.org.
